# Структура репозитория

## Принцип организации

В корне репозитория располагаются папки с кодом по лекциям.

Формат имени папки:

`lesson_<номер>_<тема>`

Например:

`lesson_4_debug` — материалы по лекции 4 «Отладка и логирование».

Внутри каждой папки находятся файлы и каталоги, повторяющие структуру реального проекта (относительно его корня).

## Пример структуры

Пример

lesson_4_debug

├─ local

│  └─ php_interface

│     └─ init.php

└─ test

   └─ log

      └─ index.php
